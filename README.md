## yeet

> *Yeet node_modules into the mass_energy() void.*

Zero dependencies. Pure Node.js. Maximum satisfaction.

```bash
npx @run-veridian/yeet
```

### Usage

```bash
yeet                      # Yeet node_modules
yeet --everything         # Yeet node_modules, .next, dist, .cache, coverage...
yeet --target .next       # Yeet a specific directory
yeet --force              # Skip confirmation
yeet --dry                # Preview without deleting
```

## roast

> *Get your code roasted by an AI.*

Brutally honest code review with adjustable savagery. Powered by Claude.

```bash
npx @run-veridian/roast src/index.ts
```

### Usage

```bash
roast src/index.ts            # Honest mode (default)
roast src/index.ts --gentle   # Constructive mentor
roast src/index.ts --savage   # Unhinged roast
roast --clipboard             # Roast from clipboard
roast src/ --yolo             # Roast entire directory
```

### Severity Levels

| Flag | Vibe |
|------|------|
| `--gentle` | Kind mentor — constructive feedback |
| *(default)* | Honest friend — direct but fair |
| `--savage` | Unhinged — pure comedic destruction |

## alibi

> *Generate convincing git history for your "sick day".*

Create backdated commits that look totally legit. For educational purposes only. Obviously.

```bash
npx @run-veridian/alibi
```

### Usage

```bash
alibi                         # Generate 8 hours of commits
alibi --hours 4               # Generate 4 hours
alibi --date 2026-01-24       # Backdate to specific date
alibi --style slacker         # 3-4 commits, casual pace
alibi --style overachiever    # 12-17 commits, maniac mode
alibi --dry                   # Preview without committing
```

### Styles

| Style | Commits | Vibe |
|-------|---------|------|
| `slacker` | 3-4 | Casual, big gaps |
| `normal` | 6-8 | Regular workday |
| `overachiever` | 12-17 | Maniac mode |

## Installation

### Run directly with npx

```bash
npx @run-veridian/yeet
npx @run-veridian/roast src/index.ts
npx @run-veridian/alibi --dry
```

### Install globally

```bash
npm install -g @run-veridian/yeet @run-veridian/roast @run-veridian/alibi
```

### Requirements

- **Node.js 18+**
- **roast** requires `ANTHROPIC_API_KEY` environment variable ([get one here](https://console.anthropic.com))
- **alibi** must be run inside a git repository
